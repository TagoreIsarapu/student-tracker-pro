# 🎓 Student Tracker Pro

A Full-Stack College Attendance Management System built using React and Supabase.

---

## 🚀 Features
- User Signup & Login Authentication
- Teacher ID Based Registration
- Branch Selection System
- Secure Profile Creation
- Attendance Management
- Cloud Database Integration

---

## 🛠 Tech Stack
- React
- TypeScript
- Supabase
- Tailwind CSS
- Vercel Deployment
